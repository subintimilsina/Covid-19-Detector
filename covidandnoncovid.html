RUN THE MODEL TO CHECK FOR ACCURACY

We used DarkNet model which was properly tuned to be accurate for our project system.DarkNet is a classifier model that forms the basis of realtime object detection system called YOLO.

DarkNet model is a typical CNN structure which has:
• a convolution layer that extracts feature from the input with the filter applied on it.
• a pooling layer to reduce the size of computational performance.
• a fully connected layer which is combined one or more times.
